# Library Management System

## Setup instruction

1. Clone the project

```bash
    git clone https://github.com/Spai12/Library-Management-System.git
```

2. Navigate to the directory
```bash
    cd LMS
```


4. Install dependencies

```bash
    npm install
```

4. Run the Backend server

```bash
    node server.js
```
5. Run the Frontend server

```bash
    npm start
```
